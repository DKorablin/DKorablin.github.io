SAL Host Dialog [Lite] (Dialog based Interface basic version)

{% include files_list.html releases=site.data.Flatbed-Dialog-Lite %}

Basic dialog based interface to quickly open any plugin with UI.

It will starts to enumerate all loaded plugins and try to open it UI by invoking IMenuItem.OnClick. It's done to simplify procedure to run simple tasks without describing user where he need to click to open an application.

It contains minimum logic and will do nothing if there is no any plugins in current folder. All launch logs will be stored to Flatbed.Dialog.Lite.log file.

Automation

Host will automate following tasks:
  1. Open all found dialog windows
  2. "Stick" windows to each other based on component StickyWindow.
  3. Save on close and restore windows positions and sizes
